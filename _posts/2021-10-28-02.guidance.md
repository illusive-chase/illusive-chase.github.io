## 第四次作业

http://dxfjgc.openjudge.cn/hw04/



### 1. 输入多个实数直到QUIT停止

变量：n, lst。

```python
n = input()
lst = []
while n != "QUIT":
    lst.append(float(n))
    n = input()
```

这样一来lst就是这些实数的列表。



### 2. 求中位数

变量：lst, half, mid。

```python
if len(lst) % 2 == 0:
    half = len(lst) // 2
    mid = (lst[half - 1] + lst[half]) / 2
else:
    half = len(lst) // 2
    mid = lst[half]
```

mid就是lst的中位数。



### 3. 输出多个数用空格分开，精确到0.001

变量：a, b, c, d。

```python
print("%.2f %.2f %.2f %.2f" % (a, b, c, d))
```



### 4. 求平均数

变量：lst, mean。

```python
mean = sum(lst) / len(lst)
```

mean就是lst的平均数。



### 5. 读入N个整数，获取其中的最大值

变量：i, N, val, maxVal。

```python
maxVal = -9999
for i in range(N):
    val = int(input())
    if val > maxVal:
        maxVal = val
```

这样最后maxVal就是其中的最大值。



### 6. 输出：精确到小数点后第t位

变量：x, t。

```python
print(f"%{t}f" % x)
```

这样输出x精确到了小数点后第t位。



### 7. 写一个判断是否是偶数的函数

```python
def is_even(x):
	if x % 2 == 0:
		return True
	else:
	    return False
```





## 第五次作业

http://dxfjgc.openjudge.cn/hw05/



### 1. 求e^x

变量：x, result。

```python
import math
result = math.exp(x)
```

这里result就是结果。



### 2. 将用空格分隔的一行正整数读入成为列表

变量：lst, N。

```python
lst = []
for N in input().split():
    lst.append(int(N))
```

这里lst就是读入的正整数的列表。



### 3. 将列表按函数的结果从小到大排序

变量：lst。

函数：F。

```python
lst.sort(key=F)
```

这里对于lst中的每个元素，假设叫x，根据F(x)的大小来排序。

即：排序完的列表中，F(x)越大的x，排在越前面。



### 4. 判断字典d是否有键值x

变量：b, x, d。

```python
b = x in d
```

b就是“字典d是否有键值x”的结果，是True或者False。



### 5. 将字符串的列表转化成整数的列表

变量：i, lst1, lst2。

```python
lst2 = []
for i in lst1:
	lst2.append(int(i))
```

这样lst2就是lst1里每个元素变成整数之后的结果。

或者：

```python
lst2 = [int(i) for i in lst1]
```



### 6. 将字典转换成列表并输出“键值:值”

变量：d, lst, key, value。

```python
lst = list(d.items())
for key, value in lst:
    print(str(key) + ":" + str(value))
```



### 7. 将字典转换成列表并按照值从小到大排序

变量：d, lst。

```python
lst = list(d.items())
lst.sort(key=lambda x:x[1])
```



### 8. 将字典转换成列表并按照键值从小到大排序

变量：d, lst。

```python
lst = list(d.items())
lst.sort(key=lambda x:x[0])
```
